- [ ] Check if all dependencies are up to date.

- [ ] Migrate existing `.js` and `.jsx` files to `.ts` and `.tsx`.

- [ ] Transition from Next.js page routing to app routing.
- [ ] Update all pages according to the new routing system.

- [ ] Replace Redux with Zustand.
- [ ] Update state management logic in all relevant components.

## Testing and Debugging
- [ ] Update tests for new components and services.
- [ ] Ensure all functionalities work correctly after changes.

## Documentation and Comments
- [ ] Update documentation to reflect project changes.
